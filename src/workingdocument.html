What is your biggest strength?

Incorporating concepts I learned in CoderGirl, I went from knowing nothing about programming, to building a project using JavaScript with React as a framework integrating Node for a database and incorporating multiple APIs. I was able to create something that showcased the hard work and effort it took to learn a new skill.
